# Red Thread / Textile / Architecture / Movement

This repository contains the project files for a DfPI Digital Skills project exploring red thread, textile systems, architecture, and movement through data-driven visualisation and Processing animation.

## Structure

- `data/csv/` - project CSV datasets and processed data tables
- `data/images_sample/` - small sample image set for reference
- `notebooks/` - Jupyter notebooks used for data collection and analysis
- `scripts/` - Python scripts
- `gh/` - Grasshopper files
- `blender/` - GitHub-safe lightweight OBJ assets
- `processing/` - Processing sketches
- `outputs/` - processed results and exported data

## Notes

Large videos and oversized 3D files are not included in the upload-ready project files. The original large Blender/OBJ files remain on the local machine; `blender/1_lightweight.obj` is included for GitHub and Processing.
